Development
===========

Virtualenv
----------

$ pip install virtualenv

$ virtualenv dev

$ source dev/bin/activate

$ pip install -r requirements.txt

$ pip install -e . # Install setup.py in development mode

...

$ deactivate

Requirements
------------

gcc, python-devel, postgresql-devel
